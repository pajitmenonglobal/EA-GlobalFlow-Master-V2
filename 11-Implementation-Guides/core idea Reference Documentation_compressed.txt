Installation Guide
Step 1: VPS Setup

bash
1. Order ForexVPS.net Standard Plan
2. Connect via Remote Desktop
3. Install IC Markets MT5
4. Create folder structure: C:\MT5_IC_Markets\ C:\EA_GlobalFlow_Bridge\ C:\EA_GlobalFlow_Data\

Step 2: Python Bridge Installation

bash
1. Install Python 3.9+
2. Extract EA_GlobalFlow_Python_Portable.zip
3. Run: pip install -r requirements.txt
4. Configure config.json with API keys

Step 3: EA Installation

bash
1. Copy EA files to: MT5\MQL5\Experts\
2. Copy indicators to: MT5\MQL5\Indicators\
3. Restart MT5
4. Attach EA to any chart
5. Load provided .set file

Step 4: Configuration
1. Set account type (NRI/Resident)
2. Configure risk parameters
3. Set trading hours
4. Enable desired markets
5. Start trading

Troubleshooting

Solution:
- Check Windows Firewall
- Verify API credentials
- Restart Python bridge
- Check named pipe permissions

2. TrueData Feed Issues

Solution:
- Verify subscription active
- Check internet connection
- Restart feed service
- Clear cache folder

3. Order Rejection

Solution:
- Check NRI restrictions
- Verify margin available
- Check OPS limit
- Review error logs
